上传到github仓库，并在线浏览网页

首先准备好你需要上传的文件夹，里面有你写好的index.html文件js和css文件等
在账号下创建一个仓库，名字为timer
来到你的目录下，我的是intro

此时已经进入文件夹，输入
git add index.html
git add style.css
git add render.js
git commit -m “first commit”

git remote add origin git@github.com:yourname/intro.
git git push -u origin master

之后显示

    就可以了，返回到你的仓库里，就可以看到了
